# layered_ventures

This is the mono-root for Layered Ventures operations, projects, and shared contracts.

- lv_ops/: doctrine + ops logs + tools (source of truth for how we work)
- projects/: modular project folders (lvcc, puremetalprints, mechanick, layered-studios)
- shared/: shared contracts/packages/assets used across projects
- archive/: historical exports and snapshots
- sandbox/: temporary scratch work (safe to delete)
